# Session Checkpoint Template

Use this template at the end of a session to update `progress.md` or to draft a new session summary.

```md
### Session YYYY-MM-DD

- Goal:
  - 
- Completed:
  - 
- Artifacts Produced:
  - 
- Blockers:
  - 
- Decisions Made:
  - 
- Next Actions:
  - 
```

## Update Checklist

- Refresh `Current Status` in `progress.md` if the phase changed
- Add newly completed work to `Completed`
- Update `In Progress`
- Record blockers or risks if they appeared
- Replace `Next 3 Tasks` with the next concrete items
- Add new entries to `decisions.md` if a real scope or strategy decision was made
- Only update `master_plan.md` if milestones or strategy changed materially
